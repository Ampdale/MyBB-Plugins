Description

Spammer Banhammer

Spammer Banhammer allows admins to quickly view a list of users that can be spammers based on:

  • Total Posts (under a certain threshold)
  • Homepage (containing links)
  • Biography (containing links)
  • Signature (containing links)
  • Usergroup (Registered and Awaiting Activation groups only!)

Moreover, you can Purge All spammers that match the following criteria: (where Y is a setting)

  1. Less than (or equal to) Y posts, links in Signature, Website and Biography
  2. Less than (or equal to) Y posts, links in Signature and Biography
  3. Less than (or equal to) Y posts, links in Signature and Website
  4. Less than (or equal to) Y posts, links in Website and Biography
  5. Less than (or equal to) Y posts and links in Signature
  6. Less than (or equal to) Y posts and links in Website
  7. Less than (or equal to) Y posts and links in Biography

The total posts required to be in the page and the profile field ID of the biography field can be configured in the plugin settings.

If you have a lot of spammers, running the Purge All matching Criteria X may cause a 500 Internal Error due to the script taking too long to execute. To avoid this error, you should adjust the setting "Maximum Users to Purge" according to your server configuration and hardware specifications (100 is an acceptable number).

Install

Upload the contents of "Upload" folder to the root of your MyBB installation.

Install & Activate "Spammer Banhammer" from the AdminCP.

Go to ACP -> Configuration -> Settings -> Spammer Banhammer and configure the settings.

The Spammer Banhammer page can be found at ModCP -> Spammer Banhammer.

Upgrade

Upgrade to 1.1

Upload the contents of "Upload" folder to the root of your MyBB installation.

Uninstall the plugin and install it back. Settings will need to be re-configured afterwards.
