Description

MyChatbox

Adds a chat box to MyBB using jQuery.

Features:

  • Permissions per group: allowed to view, post, edit, delete and commands.
  • Allow MyCode.
  • Allow IMG tags.
  • Allow video tags.
  • Chatbox logs.
  • Custom message displayed for users that cannot view the shoutbox.
  • Manage bans from Users & Groups -> MyChatbox.
  • Special commands:
    • /clear => clears all messages.
    • /clear dd/mm/yy => clears all messages since the specified date.
  • NewPoints compatibility: price per message.
  • NewPoints compatibility: groups exempt from fees.

Install

1. Upload the contents of the Upload folder to the root of your MyBB installation.

2. Go to the Admin Panel and install "MyChatbox".

3. Go to ACP -> Settings -> MyChatbox and change anything you need.

4. Go to ACP -> Themes & Templates -> Templates -> Add {mychatbox} to the template you want the chatbox to be displayed on.

Upgrade

1.1 -> 1.2

1. Upload the contents of the Upload folder to the root of your MyBB installation.

1.0 -> 1.1

1. Upload the contents of the Upload folder to the root of your MyBB installation.

2. Go to the Admin Panel -> Configuration -> Plugins page and deactivate "MyChatbox".

3. Activate it back.

Instructions

To manage bans go to ACP -> Users & Groups -> MyChatbox.
