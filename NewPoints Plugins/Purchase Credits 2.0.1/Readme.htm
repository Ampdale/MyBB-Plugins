Description

Purchase Credits

Purchase Credits allows board administrators to setup points packages on their MyBB forum to be purchased by users.

Features:

  • Multiple packages.
  • Payment types: PayPal, Payza (previously known as AlertPay) and CoinPayments
  • Logs all payments.

Payments logs can be viewed on ACP -> NewPoints -> Packages -> PayPal Logs, Payza Logs or CoinPayments Logs.

Install

Simply upload the contents of the "Upload" folder to the root of your MyBB installation.

Make sure to edit the settings for Purchase Credits!

The PayPal account should have Instant Paypal Notifcations enabled - please refer to the PayPal documents for more information on this matter.

For Payza payments, you must enter your IPN Secret Code in line 36, file credits_payza.php

For CoinPayments payments, you must enter your IPN Secret Code in line 29, file credits_coinpayments.php

Upgrade

Upgrading from 2.0 to 2.0.1:

Overwrite all files.

Upgrading from 1.0/1.1/1.2 /1.2.1 to 2.0.1:

Overwrite all files.

Upload the upgrade20.php file to the root of your forum installation and run it from your browser. Delete it afterwards.

Deactivate the plugin and activate it back (do not re-install!).

Instructions

The Purchase Credits page in your ACP can be found in ACP -> NewPoints -> Packages.
