Description

MyDownloads

Features:

  • AdminCP
    • Lots of settings, which allow you customize plenty of features.
    • Manage categories and downloads
      • Many sub-category levels
      • Permissions to view, download and submit items
    • Logs (ratings, downloads and comments)
    • Submissions - downloads that have been submitted by users and are awaiting approval
    • PayPal Logs - payment logs
    • Reported Items
    • Tags
  • Font End
    • (Optional) Most recent downloads box on index and portal
    • Home page with stats: most downloaded, most viewed, most rated
    • Many sub-category levels
    • Users can submit their own downloads (either file or download links)
    • Users can submit their own downloads
    • Users can comment downloads
    • Users can rate downloads
    • Download authors can upload (through drag and drop) multiple previews for their downloads and can set one as cover
    • Users can manage their submissions (edit and delete)
    • Users can report downloads.
    • Supports PayPal and NewPoints.
    • Users can search downloads in all categories or specific categories.
    • Users can associate tags with their downloads and filter downloads by tags.

Install

Upload Files

Simply upload the contents of the "Upload" folder to the root of your MyBB installation.

Configuration & File Permissions

Then, activate MyDownloads from the Administrator Panel and chmod the downloads directory (by default is ROOT/mydownloads/downloads) as well as the previews directory (by default is ROOT/mydownloads/previews) to 777. You can change the downloads and previews directory by changing the settings.

Template Edits

You may require to add {$mydownloads} to your index template, right before {$forums} as well as in the portal template right after {$pms}.

You also require to place a link to www.yoursite.com/mydownloads.php somewhere in the header template so users know where the downloads page is.

In addition to these, you may need to add {$mydownloads_reports} in the 'global' template right after {$unreadreports}.

Upgrade

Upgrading from 3.0/3.0.1 to 3.0.2:

Upload all files, overwriting exiting ones.

Upgrading from 2.8.1 to 3.0:

Upload all files, overwriting exiting ones.

Deactivate (NOT UNINSTALL) and Activate the plugin back.

Upload upgrade script upgrade30.php to your forum root. Run it.

DELETE IT now.



For older versions, please download the 2.8.1 ZIP from the release thread.

Instructions

Manage categories/downloads

Create category: Admin CP -> MyDownloads tab. Click Create category if you want to create a main category; If you want to create a sub category, browse an existing category and then click Create Category.

Add download: Admin CP -> MyDownloads tab. Browse the category (by clicking the category name) where you want the download to be in and then click Add download.

PayPal

In order to use the PayPal feature, you must do the following:

  • Go to the settings page and click MyDownloads.
  • Enable PayPal.
  • Enter your PayPal email address.
  • Select the currency you want to use.
  • You can choose if users may enter their PayPal email to get the money or if you want the money to go into your account.

NewPoints

You can make it so users pay points to download by enabling the NewPoints compatibility setting. Make sure NewPoints is installed, otherwise it will not work.
